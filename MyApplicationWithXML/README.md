# 결제 시스템

## 구매 혜택
### 포인트
- 적립율 
  - 한정판 품묵 2.5%
  - 정가 품목 2%
  - 할인 품목 1% (한정판 품목 제외)
- 사용 기간
  - 13개월
- 적립 동시에 사용 불가능
## 할인
### 쿠폰
- 정률제
    - 중복 할인(정액제 계산 후 할인)
        - 3% 할인(이벤트 쿠폰)
        - 2% 할인(구매 개수 제한과 가격 제한)
    - 단일 할인
        - 10% 할인(첫 구매 할인)
        - 8% 할인(두번째 구매 할인)
        - 5% 할인(세번째 구매 할인)
- 정액제
    - 3, 5, 7, 9000원
    - 10, 11000원(이벤트)
    - 사용 기간
      - 일반 : 21일 
      - 이벤트: 12일
### 계산 방법
먼저 포인트로 할인 후 정액제 계산 & 정률제 계산 그리고 나머지 결제 금액으로 포인트 적립
